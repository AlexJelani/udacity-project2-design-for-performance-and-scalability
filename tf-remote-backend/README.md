# Terraform Remote Backend Setup

This project demonstrates how to configure and use an S3 remote backend for Terraform state files.

## Project Structure
tf-remote-backend/
├── backend.tf
├── main.tf
├── variables.tf
├── outputs.tf
└── README.md

## Prerequisites
- AWS CLI configured
- Terraform installed
- AWS account with appropriate permissions

## Steps to Execute

1. Initialize Terraform:

terraform init
